Instructions on Use

1. Type in the starting address into the Start Location textbox

2. When you have entered a space an autocomplete menu will appear underneath(make sure the address is within the allowed boundaries !!!)

3. Use the arrow keys or the mouse to select one of the options

4. Repeat with the End Location textbox

5. Click on Calculate to get the shortest path between them

6. An interactive map will show with the shortest path highlighted

The red tag shows the start location and the blue tag shows the end location

Click on the path to get its distance and time to walk

Want avoid high crime?

Repeat as above but tick the Avoid High Crime checkbox

A choropleth map will appear with two routes

The green route is the actual shortest path and the black route is the shortest path which avoids high crime areas

Click on the paths to get their distance, times and crime rating

A larger crime rating shows a more dangerous route

General Use

Click the Swap button to swap the contents of the Start and End location textboxes

Click the Clear button next to the textbox which you want to clear the contents of
